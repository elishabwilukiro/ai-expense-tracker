# Personal Expense Tracker with AI Insights

## Technology Used
- Backend: Laravel + JWT Authentication
- Frontend: React
- Database: MySQL
- AI: OpenAI API
- Charts: Chart.js

### Backend
- composer install
- php artisan migrate
- php artisan db:seed
- php artisan serve

### Frontend
- npm install
- npm start
